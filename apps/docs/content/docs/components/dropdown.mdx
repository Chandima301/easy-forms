Stores `value | null` (the selected option's `value`).

<LiveForm
	schema={{ groups: [{ questions: [
		{ key: 'country', label: 'Country', control: 'dropdown', placeholder: 'Pick a country', clearable: true, options: [
			{ value: 'us', label: 'United States' },
			{ value: 'gb', label: 'United Kingdom' },
			{ value: 'lk', label: 'Sri Lanka' },
		], validators: { required: true } },
	] }] }}
	initialValues={{ country: null }}
/>

## Dynamic options

Options frequently depend on another field (country → region). Drive them with a
`propsDependsOn` rule — see [Dynamic props](/docs/dynamic/props-depends-on).
