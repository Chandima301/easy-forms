The biggest change when adopting Easy Forms is conceptual: you stop wiring inputs
imperatively and start **describing** the form as data. There is no `useForm`, `register`,
or `Controller`.

<Callout type="info">
	Migrate incrementally. Easy Forms renders into any component, so you can convert one form
	at a time and leave the rest as they are.
</Callout>
