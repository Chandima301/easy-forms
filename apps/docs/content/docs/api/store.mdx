```ts
import { createFormStore } from '@easy-forms/core';

const store = createFormStore({ initialValues, validationMode });
```

Most apps never call this — `<Form>` creates a store internally. Create one yourself to
drive values programmatically or share state across components, then pass it to
`<Form store={store}>`.
